# README

API implementation using [endpoints][1] that follows [JSON API specs][2].

## Steps

1. `git clone https://github.com/tenzan/node-api-pg`

2. `npm install`

3. `npm start`

4. APIs:
 - `http://localhost:8080/v1`
 - `http://localhost:8080/v1/posts`
 - `http://localhost:8080/v1/tags`
